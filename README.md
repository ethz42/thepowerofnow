# 当下的力量 - 深度学习工具

一个帮助读者深入理解《当下的力量》的互动式学习平台。

## 功能特点

- 📖 **理解全书**：10章核心命题 + 论证地图 + 20个反常识洞见
- 🎯 **学习概念**：5个核心概念，自由选择学习顺序
- 🧰 **工具箱**：互动游戏和实践练习

## 技术栈

- 纯HTML/CSS/JavaScript
- 无框架依赖
- 本地存储用户进度

## 部署

### 方式一：Vercel（推荐）
1. 安装 Vercel CLI：`npm i -g vercel`
2. 运行：`vercel`
3. 按提示完成部署

### 方式二：Netlify
1. 拖拽整个文件夹到 [Netlify Drop](https://app.netlify.com/drop)
2. 即时部署完成

### 方式三：GitHub Pages
1. 推送代码到 GitHub
2. 在仓库设置中启用 GitHub Pages
3. 选择主分支作为来源

## 本地运行

直接用浏览器打开 `index.html` 文件即可。
